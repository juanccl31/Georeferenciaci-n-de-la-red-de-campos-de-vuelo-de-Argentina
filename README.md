# Georeferenciaci-n-de-la-red-de-campos-de-vuelo-de-Argentina
Básicamente la actividad se centró en el análisis actual del MADHEL y de los diferentes AIPs de la Republica Argentina, generando una base de datos propia para luego desarrollar una herramienta que, mediante la aplicación de ciertos filtros permitiera la visualización de la red bajo diferentes criterios de análisis.
